windows系统安装

针对本人MAC下的虚拟机win7系统

  • 安装前备份
  • OS下载：个人或者微软下载中心，注意微软下载中心的”Oct 2018"版本有问题，应选”April 2018“
  • 使用老毛桃制作windows启动盘
  • 运行win7激活工具

工具

必选

  • 驱动精灵万能网卡版
  • 安装VIM，并配置好;
  • 安装mtux，并配置好；
  • 安装git/git GUI及汉化包，并配置好git;
  • 为便于git clone git@github.com:xxx/yyy时无需密码验证, 进行github下的ssh配置

可选

  • 烧录工具：stc-isp-15xx-v6.86S
  • SD卡格式化：SDFormatter
  • SD卡备份与恢复：Win32DiskImager
  • windows下的包管理工具：scoop、baulk
  • everything
  • autohotkey
  • Total Commander
  • win10截图：snippingtool

虚拟机

v1

存放位置

  • 正版win7 64位旗舰版
  • 已映射下载快捷方式：方便与宿主机互访；
  • 已映射work快捷方式: 工作文件与绿色软件；
  • 已安装TeamViewer

v2

存放位置，在v1基础上，用于STC下的开发工作，建议安装最新v5版本

  • 执行MDKxxx.exe安装IDE,缺省安装了ARM;
  • 手动安装c51，期间出现的所有提示都选skip;
  • 运行注册机
